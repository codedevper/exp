## wp language
wp language core list -- list available core language packs
wp language core activate <language> -- activate a core language
wp language core install <language>... -- install core language pack(s)
wp language core uninstall <language> -- uninstall a core language pack
wp language core update -- update core language packs
wp language plugin list <plugin> -- list available language packs for a plugin
wp language plugin activate <plugin> <language> -- activate a plugin language
wp language plugin install <plugin> <language>... -- install plugin language pack(s)
wp language plugin uninstall <plugin> <language> -- uninstall a plugin language
wp language plugin update <plugin> -- update plugin language packs
wp language theme list <theme> -- list available language packs for a theme
wp language theme activate <theme> <language> -- activate a theme language
wp language theme install <theme> <language>... -- install theme language pack(s)
wp language theme uninstall <theme> <language> -- uninstall a theme language
wp language theme update <theme> -- update theme language packs
