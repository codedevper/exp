## wp menu
wp menu list -- list registered menus (locations)
wp menu create <name> -- create a new menu
wp menu delete <menu>... -- delete one or more menus
wp menu assign <menu> <location> -- assign a menu to a theme location
wp menu location list -- list available theme menu locations
wp menu item list <menu> -- list items in a menu
wp menu item add <menu> [--title=<title>] [--link=<url>] [--description=<desc>] -- add an item
wp menu item update <db_id> [--title=<title>] [--link=<url>] -- update a menu item
wp menu item delete <db_id> -- delete a menu item
