## wp widget
wp widget list <sidebar-id> [--fields=<fields>] -- list widgets in a sidebar
wp widget add <name> <sidebar-id> [--position=<n>] [--<field>=<value>] -- add a widget
wp widget update <widget-id> [--<field>=<value>] -- update a widget's settings
wp widget delete <widget-id>... -- delete widget(s)
wp widget reset <sidebar-id>... [--all] -- remove all widgets from a sidebar
wp widget deactivate <widget-id> -- deactivate a widget (move to inactive)
wp widget move <widget-id> <sidebar-id> [--position=<n>] -- move a widget to another sidebar
