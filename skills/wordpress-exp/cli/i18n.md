## wp i18n
wp i18n make-pot <source> [<target>] -- create a POT file from the project
wp i18n make-json <source> [<target>] [--purge] -- build JSON translation files from PHP files
wp i18n make-mo <source> [<target>] -- create MO files from PO files
wp i18n update-po <source> [<target>] -- update PO files from POT file
wp i18n make-php <source> [<target>] -- create PHP translation files from JSON files
