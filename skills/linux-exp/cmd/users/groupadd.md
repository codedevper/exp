## groupadd
sudo groupadd <group> -- create a new group
sudo groupadd --system <group> -- create a system group
sudo groupadd -g <gid> <group> -- create group with specific GID
sudo groupadd -r <group> -- create a system group (alternative)
sudo groupadd -f <group> -- exit silently if group exists
sudo groupadd -p <password> <group> -- set group password
