## snap
sudo snap install <snap> -- install a snap package
sudo snap remove <snap> -- remove a snap package
sudo snap list -- list installed snaps
sudo snap refresh <snap> -- update a snap
sudo snap revert <snap> -- revert to previous version
sudo snap disable <snap> -- disable a snap
sudo snap enable <snap> -- enable a snap
snap find <query> -- search for snaps
snap info <snap> -- show snap details
snap changes -- list recent snap changes
snap services -- list snap services
