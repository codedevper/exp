## adduser
sudo adduser <username> -- create a new user with home dir
sudo adduser --system <username> -- create a system user
sudo adduser --no-create-home <username> -- create user without home dir
sudo adduser --ingroup <group> <username> -- assign to specific group
sudo adduser --uid <uid> <username> -- create user with specific UID
sudo adduser --disabled-password <username> -- create user without password
sudo adduser --shell <shell> <username> -- set login shell
