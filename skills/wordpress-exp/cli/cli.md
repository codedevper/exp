## wp cli
wp cli info -- print WP-CLI info (version, PHP, server)
wp cli update -- update WP-CLI to the latest version
wp cli check-update -- check for WP-CLI update availability
wp cli version -- print WP-CLI version
wp cli param-dump -- dump the list of registered WP-CLI params
wp cli alias list -- list registered aliases
wp cli alias add <name> <value> -- add a new alias
wp cli alias update <name> <value> -- update an alias
wp cli alias delete <name> -- delete an alias
wp cli has-command <command> -- check whether a WP-CLI command exists
wp cli cmd-dump -- dump the list of registered commands
