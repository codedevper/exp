## wp embed
wp embed fetch <url> [<post_id>] -- fetch and cache an oEmbed URL
wp embed handler list [--field=<field>] [--format=<format>] -- list registered embed handlers
wp embed provider list [--field=<field>] [--format=<format>] -- list registered oEmbed providers
wp embed cache clear [<url>] [<post_id>] -- clear the embed cache
